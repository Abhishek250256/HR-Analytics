# HR-Analytics

## Project Overview

Welcome to the **Atliq Service HR Analytics Dashboard** repository! This initiative offers an interactive Power BI dashboard designed to assist Atliq Service, a growing company, in examining employee performance, attendance, work-from-home (WFH) status, and leave trends. The objective is to provide valuable insights in these critical areas to aid in decision-making and enhance overall HR management.

## Features
- Interactive Power BI dashboards for various HR metrics
- Analysis of employee performance and productivity
- Monitoring of employee presence and attendance
- Insights into WFH trends and patterns
- Detailed leave analysis and trends
